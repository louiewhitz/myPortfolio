Projects

The Check-In: An Easy to Use App for Care Collaboration

LearningFuze | Aug 2022 – Dec 2022

The Check-In is a user-friendly web application that helps you manage and keep track of important events and information for people you care about. It features a secure sign-in/register page for user authentication and allows you to add events with details such as title, description, and images which are stored securely on AWS-S3. The Timeline Page provides an overview of all events and images, with the option to edit or delete events. The View Photos page displays images in full view and the Scheduling/Calendar page features a large, customizable calendar with the option to schedule events with a title and time. The Check-In offers a convenient and easy solution for managing important events and information.

Capture Nasa: An App that teaches you about the Stars

LearningFuze | Aug 2022 – Dec 2022

Capture NASA is a simple, solo-built web application that allows users to view NASA's image or video of the day, save their favorites, and search past images/videos. It's designed for optimal viewing on any device, using custom HTML and CSS for structure and design. The application uses AJAX requests to NASA's APOD API to fetch the image/video of the day and an inspiring quote from Quotable API. The project is written in ES6 but can easily be converted to ES5 if desired.
