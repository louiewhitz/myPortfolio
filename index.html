The Check-In

The Check-In is a web app built with PERN (PostgreSQL, Express, React, Node) stack. It lets users share information and schedule events for people they care about. The app has a sign-in/register page using JWT and Argon2 for secure authentication. Users can add events with a type, title, description, and optional image (stored in AWS S3) on the Add Event page. The Timeline Page shows all events and images, allowing for editing or deletion. The View Photos page displays all photos in full view. The Scheduling/Calendar page has a React Big Calendar with month, week, day, and agenda views and a datetime picker to schedule events with a title and time.

Deployed App: https://the-check-in.louisawhitaker.com/

Github: https://github.com/louiewhitz/the-Check-in

Capture Nasa

Capture Nasa is a web application that allows users to view NASA's image or video of the day, save their favorite image or video to a favorites page, and search for past images or videos.This application is fully responsive design for optimal viewing on any device. The app was built using simple front end technologies implementing AJAX requests to NASA's APOD API in order to vie NASA's image and/or video of the day and search by day, month and year in the past. Furthermore, on the homepage, each time the page loads, another AJAX request is sent to Quotable API in order to fetch an inspiring quote.

GitHub Deployment Link: https://louiewhitz.github.io/capture-nasa/
